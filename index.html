MDM vs DWH vs CDI vs Data Quality
MDM (Master Data Management)
  • Система управления мастер-данными
  • Ее цель - обеспечить единство представления данных, полученных из разных систем
    • DWH (Data Warehouse)

  • Хранилище данных куда стекается куча разной информации из разных мест.
    Данные хранятся разрозненно
    • CDP (Customer Data Platform)
  • Платформа хранения клиентских данных
    • CDI (Customer Data Integration)
  • Мастер-система клиентских данных (только клиентских)
